# 📈 株シグナル通知ボット v3（LINE通知版）

毎朝8時・前場10:30・後場13:30に、日本株のシグナルを **LINEに直接通知** するボットです。

- GitHub Actions上で動作 → **サーバー・VPS不要**
- LINE Notifyを使用 → **パスワード設定不要、トークン1つだけ**
- 必要なSecrets（秘密情報）は **2つだけ**

## セットアップ手順

### ステップ1：LINE Notifyのトークンを取得する

**① ブラウザでサイトを開く**

SafariまたはChromeで以下のURLにアクセスします。

```
https://notify-bot.line.me/ja/
```

**② LINEアカウントでログインする**

右上の「ログイン」をタップ → LINEのメールアドレスとパスワードを入力 → ログイン

**③ トークンを発行する**

ログイン後、右上に自分の名前が表示されます。

名前をタップ →「マイページ」→ 少し下にスクロール →「トークンを発行する」ボタンをタップ

以下のように入力します。

| 項目 | 入力内容 |
|------|---------|
| トークン名 | `株ボット`（なんでもOK） |
| 通知を送るトークルーム | `1:1でLINE Notifyから通知を受け取る` を選択 |

「発行する」ボタンをタップ

**④ トークンをコピーして保存する**

英数字の文字列（例：`aBcDeFgHiJkLmNoPqRsTuVwXyZ12345`）が表示されます。

> ⚠️ **このトークンは今しか表示されません。** 必ずメモ帳や別のアプリにコピーしてください。「閉じる」を押すと二度と確認できなくなります。

**⑤ LINEアプリで確認する**

iPhoneのLINEアプリを開くと、「LINE Notify」というアカウントが友達に追加されています。ここにボットからの通知が届きます。

---

### ステップ2：J-Quantsのトークンを取得する

J-Quantsは日本取引所グループが提供する無料の株価データAPIです。

**① サイトを開く**

```
https://jpx-jquants.com
```

**② 無料登録する**

「新規登録」→ メールアドレス・パスワードを入力 → 登録メールのリンクをクリックして認証

**③ トークンを発行する**

ログイン後、右上のアカウントアイコン →「APIトークン発行」

「リフレッシュトークン」という文字列が表示されます。これをコピーして保存。

---

### ステップ3：GitHubにリポジトリを作成してファイルをアップロードする

**① GitHubアカウントを作成する（まだの場合）**

```
https://github.com
```

「Sign up」から登録（無料）

**② 新しいリポジトリを作る**

ログイン後、右上の「＋」→「New repository」

| 項目 | 入力内容 |
|------|---------|
| Repository name | `stock-signal-bot` |
| Public / Private | どちらでもOK（Privateを推奨） |

「Create repository」をクリック

**③ ファイルをアップロードする**

ダウンロードしたZIPを解凍します。

リポジトリのページで「uploading an existing file」リンクをクリック

解凍したフォルダの中身を**全て選択してドラッグ＆ドロップ**します。

> ⚠️ `.github` フォルダも必ずアップロードしてください（隠しフォルダなので見えにくい場合があります）。

「Commit changes」をクリック

---

### ステップ4：GitHub Secretsに2つ登録する

SecretとはGitHubの「金庫」です。パスワードなどの秘密情報をここに保管することで、コードに直接書かずに安全に使えます。

**① Secretsのページを開く**

リポジトリのページ → 上部の「Settings」タブ → 左メニューの「Secrets and variables」→「Actions」

**② 1つ目を登録する**

「New repository secret」ボタンをクリック

```
Name（名前）  : JQUANTS_REFRESH_TOKEN
Secret（値）  : ステップ2でコピーしたJ-Quantsのトークン
```

「Add secret」をクリック

**③ 2つ目を登録する**

再度「New repository secret」をクリック

```
Name（名前）  : LINE_NOTIFY_TOKEN
Secret（値）  : ステップ1でコピーしたLINE Notifyのトークン
```

「Add secret」をクリック

登録後、以下のように表示されれば完了です。

```
JQUANTS_REFRESH_TOKEN    Updated just now
LINE_NOTIFY_TOKEN        Updated just now
```

> ℹ️ FinnhubのAPIキーをお持ちの場合は `FINNHUB_API_KEY` も登録すると、重要経済指標の警告機能が有効になります。なくても動作します。

---

### ステップ5：動作テストをする

**① Actionsタブを開く**

リポジトリページ上部の「Actions」タブをクリック

**② 手動実行する**

左側のリストから「📈 朝のシグナル通知（8:00）」をクリック

右側の「Run workflow」ボタン →「Run workflow」をクリック

**③ 結果を確認する**

1〜3分後、緑のチェックマーク✅が表示されれば成功です。

iPhoneのLINEアプリを開いて「LINE Notify」からメッセージが届いていれば完璧です。

赤い❌が表示された場合は、ジョブをクリックするとエラーの詳細が見えます。

## 通知スケジュール

| 時刻 | 内容 | 送信条件 |
|------|------|---------|
| 毎朝 8:00 | 本日の注目銘柄（最大5件） | 毎日送信 |
| 前場 10:30 | 朝の銘柄の状況変化 | 変化があった場合のみ |
| 後場 13:30 | 変化の確認＋新規候補銘柄 | 毎日送信 |

---

## 状況変化の通知パターン

| 通知 | 条件 | 意味 |
|------|------|------|
| 📈 強化シグナル | MACD GC + RSI上昇 + 出来高増加 | 継続/追加エントリー推奨 |
| ⚠️ 弱体化シグナル | 目標価格接近 or RSI過熱 | 利確を検討 |
| 🚨 撤退シグナル | 損切りライン到達 or RSI75超 | 損切り/見送り推奨 |
| （通知なし） | 変化なし | 前場はスキップ |

