# 📈 株シグナル Bot v5

毎朝8時に日本株の注目銘柄をスクリーニングしてDiscordに通知するBotです。

---

## ✅ v5 での主な変更点

| 項目 | v4（旧） | v5（新） |
|------|----------|----------|
| データ取得 | J-Quants API（全4000銘柄を1件ずつ） | yfinance（注目50銘柄を一括取得） |
| 実行時間 | 30〜60分 | **5分以内** |
| レート制限 | あり（0.2秒スリープ必要） | **なし** |
| 必要なSecrets | `JQUANTS_API_KEY` | **不要**（削除可） |
| 銘柄リスト | 毎回全銘柄から動的抽出 | 事前選定50銘柄 + **月次自動更新** |

## 🚀 セットアップ（初回のみ）

### 1. GitHubリポジトリを作成してファイルをアップロード

```
stock-signal-bot/
├── main.py
├── screener.py
├── scorer.py
├── notifier.py
├── monitor.py
├── market_context.py
├── requirements.txt
└── .github/
    └── workflows/
        ├── morning_signal.yml
        ├── midmorning_monitor.yml
        ├── afternoon_signal.yml
        └── monthly_watchlist_update.yml   ← v5で追加
```

### 2. GitHub Secrets に登録するもの

Settings → Secrets and variables → Actions → New repository secret

| Secret名 | 値 | 必須 |
|----------|-----|------|
| `DISCORD_WEBHOOK_URL` | DiscordのWebhook URL | ✅ 必須 |
| `FINNHUB_API_KEY` | Finnhub API key（無料） | 任意（重要イベント検知用） |

> **v5から `JQUANTS_API_KEY` は不要です。** 登録済みの場合は削除してOKです。

### 3. GitHub Actionsの権限設定（watchlist自動更新に必要）

Settings → Actions → General → Workflow permissions  
→ **「Read and write permissions」を選択して保存**

## 📋 注目50銘柄リストについて

### 選定基準

プロのクオンツが重視する5つの観点で選別：

1. **流動性** — 日次売買代金が安定して高い（薄い板に引っかからない）
2. **価格帯** — 500〜10,000円（個人投資家が売買しやすい）
3. **セクター分散** — 1セクター最大8銘柄に制限
4. **トレンド性** — 52週高値比 + 移動平均乖離でスコア化
5. **機関注目度** — 日経225・JPX400・TOPIX100構成銘柄を優先

### セクター構成（デフォルト）

| セクター | 銘柄数 |
|----------|--------|
| 自動車 | 5銘柄 |
| 銀行・金融 | 5銘柄 |
| 商社 | 5銘柄 |
| 半導体・電子部品 | 7銘柄 |
| 医薬品 | 4銘柄 |
| テクノロジー・通信 | 6銘柄 |
| 小売・消費財 | 4銘柄 |
| その他 | 14銘柄 |

### 月次自動更新の仕組み

`monthly_watchlist_update.yml` が**毎月1日 朝7:00 JST**に自動実行：

1. デフォルト50銘柄 + 追加候補10銘柄（計60銘柄）を対象に評価
2. 過去60日の売買代金・トレンド・ボラティリティをスコア化
3. セクター分散を保ちながら上位50銘柄を選定
4. `watchlist.json` をリポジトリに自動コミット・プッシュ

**手動でも実行可能：**  
Actions → 📋 月次 watchlist 自動更新 → Run workflow

## ⏰ 実行スケジュール

| 時刻 | 内容 |
|------|------|
| 毎朝 8:00 | 注目50銘柄をスクリーニング → Discord通知 |
| 毎日 10:30 | 朝の銘柄を監視 → 変化あれば通知 |
| 毎日 13:30 | 後場シグナル → 常に通知 |
| 毎月 1日 7:00 | watchlist.json を自動更新 |

## 📊 スコアリング基準

| 指標 | 配点 |
|------|------|
| 市場環境（日経・ドル円・SP500・ナスダック） | 最大20点 |
| 出来高比率（20日平均比） | 最大20点 |
| 価格変化率（前日比） | 最大15点 |
| RSI（50〜70が理想） | 最大15点 |
| MACD（ゴールデンクロス） | 最大15点 |
| ボリンジャーバンド位置 | 最大10点 |
| 移動平均パーフェクトオーダー | 最大5点 |
| **合計** | **100点** |

スコア50点以上の銘柄のみ通知します。
